---
name: bmad-sm
description: BMAD Scrum Master role for story drafting or revision with gate-based completion.
---

1. Read `AGENTS.override.md`.
2. Read `.bmadx/state/sprint-status.path` and the active sprint-status file.
3. Read `.codex/agents/sm.toml`.
4. Read `.bmadx/state/role-map.json` and load the BMAD SM agent markdown.
5. Follow the BMAD workflow as closely as possible.
6. Create or revise the actual BMAD story file in the `story_location` path from sprint-status.
7. Do not declare the phase complete unless PM/PO approval files exist and the story gate can pass.
